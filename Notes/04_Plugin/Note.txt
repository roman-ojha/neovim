=> So some examples of 'packer' are in :
  -> https://github.com/LunarVim/Neovim-from-scratch/blob/03-plugins/lua/user/plugins.lua

=> Command:
  -> :PackerSync (install/update plugins)
  -> :PakerCompile (compile plugins)
  -> :PackerClean (clean plugins)
  -> :PackerInstall <plugin> (install plugin) 
  -> :PackerUpdate <plugin> (update plugin)
  -> :PackerStatus (status of plugins)
  -> etc...
